## What is this

This repository contains the academic research program around the
VAIOS protocol — an open protocol for AI-native organizational intelligence.

The research program produces:
- **Academic papers** — conceptual, design science, and empirical research
  on governance architectures for AI-native organizations
- **Terminology** — canonical definitions of concepts originating from the
  VAIOS ecosystem
- **Intellectual property registry** — documented concepts with dates of
  articulation and evidence of prior art
- **Evidence** — metrics and case data from the Dorano implementation

## Relationship with VAIOS Protocol

The VAIOS protocol specification lives at
[fabio-dorano/vaios](https://github.com/fabio-dorano/vaios) (Apache 2.0).

This research repository contains proprietary intellectual property
(terminology, frameworks, analysis) built on top of the open protocol.
The distinction matters: the protocol is open; the research is protected.

## Structure

```
terminology/     — Canonical term definitions
papers/          — Academic papers (in progress and published)
working-papers/  — Position papers, whitepapers, preprints
evidence/        — Metrics and case data
ip-registry/     — Intellectual property documentation
checklists/      — Pre-publication quality checks
```
